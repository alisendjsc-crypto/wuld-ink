What it does

The library takes the recurring objection patterns that surface in real antinatalist and efilist discourse — in academic literature, in social-media exchanges, in podcast appearances, in comment threads — and organizes them as a five-tier taxonomy from emotional/reflexive shorthand at Tier 1 through genuine philosophical engagement at Tier 4 to meta-objections at Tier 5. Each entry pairs the rhetorical move with the psychological mechanism that drives it, the structural deconstruction that disarms it, and the archetype-aware variants that match the interlocutor’s posture (sophisticate / defender / drifter / blended).

The Real-World Examples (RWE) surface attests each objection with documented deployments — 136 instances harvested from primary sources, attached to specific speakers, archetypes, and rhetorical context.

How to use it

The library ships as a single self-contained HTML application — no build step, no server, works offline. Three surfaces behind an outer hash router:

Library → #/library Real-world examples → #/rwe Coda → #/coda

Direct deep-link to a specific objection’s attested deployments: library.wuld.ink/#/rwe/<objection-id>. The library overview surface is sidebar-navigated internally; per-objection deep-linking inside #/library is not externally addressable (state-driven, not hash-driven).

Provenance

The library substrate is maintained in a separate repository at github.com/alisendjsc-crypto/efilist-argument-library. The v3.7.1 stable release ships as combined.html (single-file distribution, md5 dd2abd01a43c2f173c98aa1b8c88bcbb, 2,234,272 bytes) under dual license: CC-BY-4.0 (content) + MIT (code).

The substrate is frozen at canon terminus. Content mutations require a programmatic invariant-derivation harness gate; the md5 is the integrity contract. The wuld.ink umbrella treats the library as an independent reference application linked from this page; cross-link grammar from glossary entries lands on the RWE surface (#/rwe/<id>) where the attested deployments live.
